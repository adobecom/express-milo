Interactive Visual Comparison

Click and drag on either image to select an area for detailed comparison

How to use:
  1. Click and drag on either the reference or website image to select an area
  2. The selection will be synchronized across both images
  3. Click "Compare Selected Area" to see a detailed comparison of just that region
  4. Use "Clear Selection" to start over
